# 🤖 **ai‑warevo**
**The Future of Software, Prompted into Existence** ✨

Welcome to **ai‑warevo** — an experimental development lab where not a single line of code has been written manually by a human. We explore the boundaries of modern LLMs (Large Language Models), creating software entirely through AI generation. 🧠🚀

## 🎯 **Our Concept**
We believe the future of development lies in the symbiosis of human intelligence and AI capabilities. Our mission: 🎯
* Prove the effectiveness of AI in industrial‑grade code creation. ✅
* Develop a “architecture‑via‑prompt” methodology. 🗣️
* Build tools for iterative generation and testing of systems. 🔬

## 🛠 **How We Work**
1. **Design** 📐: humans create detailed specifications — architecture, API, business logic, UI kits.
2. **Generation** 🤖: AI (Claude, GPT, Gemini) generates code based on prompts.
3. **Validation** ✅: humans verify correctness, security, and compliance with requirements.
4. **Iteration** 🔁: refine prompts to improve results (instead of editing code).
5. **Automation** 🤹: CI/CD pipelines for building, testing, and deployment.

## 🧪 **Tech Stack (AI‑Driven & Multi‑Modal)**
The AI developer orchestrates a diverse and cutting‑edge ecosystem:

### Web: Frontend & UI 🌐
* **Next.js ⚛️** — React framework with built‑in SSR/SSG, API routes, and image optimization; enables fast, SEO‑friendly web apps.
* **Zustand 🧲** — minimal, unopinionated state management library for React; simple and performant alternative to Redux.
* **Tailwind CSS 🎨** — utility‑first CSS framework; enables rapid UI development with low‑level utility classes.
* **Bun 🍪** — fast all‑in‑one toolkit for JavaScript/TypeScript; includes runtime, bundler, and package manager (faster than Node.js + npm/yarn).

### Desktop: Frontend & UI 💻
* **WPF (Windows Presentation Foundation) 🖼️** — UI framework for building rich desktop client applications on Windows.
* **CommunityToolkit.Mvvm 🎯** — .NET library with MVVM helpers, behaviors, and services; streamlines WPF app development by providing ready‑to‑use components for data binding, commands, and messaging.

### Persistence 💾
* **SQLite 🥄** — embedded, serverless, self‑contained SQL database; ideal for prototypes, microservices, and mobile apps.
* **PostgreSQL 🐘** — open‑source, advanced relational database; supports JSON, full‑text search, geospatial data, and ACID transactions.
* **pgAdmin 👮** — web‑based administration tool for PostgreSQL; provides GUI for managing databases, users, roles, and queries.

### Backend & API: C# Stack 🧑‍💻
* **C#** — primary language for enterprise backend services and Windows applications.
* **ASP.NET Core 🏗️** — framework for building robust, scalable web APIs and microservices (enterprise logic, REST/gRPC).
* **Marten 🗃️** — document database and event store for .NET, built on PostgreSQL; enables Event Sourcing and CQRS patterns.
* **Entity Framework Core 🔌** — ORM for .NET; simplifies database interactions with SQL Server, PostgreSQL, etc.
* **MediatR 📡** — implements the Mediator pattern in .NET; decouples components and simplifies command/query handling.
* **FluentValidation 🛡️** — validation library for .NET; provides a fluent interface to define complex validation rules for objects.

### Backend & API: Python Stack 🐍
* **Python** — main language for backend microservices, data processing, and ML workflows.
* **FastAPI ⚡** — modern Python web framework for building REST and WebSocket APIs with automatic OpenAPI/Swagger docs.
* **Uvicorn 💨** — lightning‑fast ASGI server to run FastAPI applications.
* **asyncio 🔄** — Python library for writing concurrent code using async/await; enables high‑performance I/O‑bound applications.
* **SQLAlchemy 🗃️** — SQL toolkit and ORM for Python; provides a full suite of enterprise‑level persistence patterns.

### Machine Learning & AI 🤖🧠
* **PyTorch 🦜** — open‑source ML framework; used for developing and training deep learning models (NLP, computer vision, etc.).
* **ML 🔮** — integration of ML models into production pipelines for classification, forecasting, anomaly detection, and content generation.

### Infrastructure & DevOps ☸️
* **Docker 🐳** — platform for developing, shipping, and running applications in containers; ensures consistency across environments.
* **CI/CD 🔄** — automation of build, test, and deployment pipelines; implemented with GitHub Actions, GitLab CI, or Azure DevOps.
* **Redis 🟥** — in‑memory data store used as a database, cache, and message broker; supports sessions, queues, and real‑time features.
* **RabbitMQ 🐰** — message broker for asynchronous communication between services; supports AMQP, MQTT, STOMP; includes **Web Management Plugin** 🌐 for monitoring queues, exchanges, and connections via web UI.

### Monitoring & Observability 📊
* **Prometheus 🕵️** — monitoring system and time‑series database; collects metrics from instrumented targets, stores time‑series data, and triggers alerts.
* **Grafana 📈** — visualization and analytics platform; creates rich dashboards for metrics from Prometheus, PostgreSQL, and other sources.
* **ELK Stack (Elasticsearch + Logstash + Kibana) 🔎** — unified logging solution:
  * **Elasticsearch 🔍** — distributed search and analytics engine; stores and indexes logs.
  * **Logstash 📦** — data processing pipeline; collects, transforms, and sends logs to Elasticsearch.
  * **Kibana 📊** — data visualization and exploration UI; creates dashboards and analyzes logs in real time.

### Developer Tools & APIs 🛠️
* **Claude, Cursor, Qwen 🤖** — AI‑powered coding assistants and IDEs; enhance developer productivity with code generation, refactoring, and debugging.
* **Helix 🦅** — fast, modern text editor built on Rust; designed for high performance and extensibility.
* **EventSub 📣** — Twitch API service for real‑time event notifications (e.g., when a streamer goes live or receives a donation).
* **Twitch Dev Rig / Extension APIs 🎮** — tools and APIs for building interactive Twitch extensions (overlays, panels, components).
* **Win32 API 🪟** — core set of Windows OS interfaces; used for low‑level system operations, GUI components, and legacy integrations.
* **GitHub Copilot 🤹** — AI pair programmer that suggests code and entire functions in real time; integrates with popular IDEs.
